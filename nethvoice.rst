=========
NethVoice
=========

NethVoice is a comprehensive communication system that integrates voice, video, and mobile capabilities.
The NethVoice suite is divided into two main applications:

- NethVoice Proxy : Manages SIP and RTP connections.
- NethVoice : Provides VoIP telephony services

All documentation is available at: `NethVoice Documentation <https://docs.nethvoice.com/>`_

# WebTop-5: disabilitare i destinatari suggeriti e salvati automaticamente in composizione mail

Puoi disabilitare il suggerimento degli indirizzi email salvati automaticamente durante la composizione di un messaggio.

Aggiungi questa impostazione globale nel pannello di amministrazione per il servizio `com.sonicle.webtop.core`:

`recipient.provider.auto.enabled = false`

Il valore predefinito è `true`.

:::note
Si tratta di un'impostazione globale, quindi la modifica si applica a tutti gli utenti.
:::

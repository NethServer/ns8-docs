# WebTop-5: configurare il protocollo `mailto:` sul browser

Quando colleghi `mailto:` a WebTop, facendo clic su un indirizzo email in una pagina web si apre nel browser una finestra di composizione già compilata.

Puoi attivarlo dal banner che compare dopo il login oppure da **User settings** → **Mail** → **Advanced**.

Se usi Google Chrome, WebTop richiede una conferma e Windows può anche chiederti di impostare il browser come app email predefinita.

Se in passato hai già usato un'altra app web per `mailto:`, apri `chrome://settings/handlers` e scegli di nuovo il gestore predefinito.

Firefox mostra un proprio messaggio di conferma. Poi imposta l'applicazione email predefinita da **Settings** → **General** → **Applications**.

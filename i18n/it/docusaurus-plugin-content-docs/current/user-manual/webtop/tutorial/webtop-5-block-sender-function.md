# WebTop-5: usare la funzione di blocca mittente

Fai clic destro sull'indirizzo del mittente di un messaggio aperto e seleziona **Blocca mittente**.

WebTop crea un filtro personale per la posta in arrivo che sposta tutti i messaggi futuri di quel mittente direttamente in **Posta indesiderata**.

:::note
Queste regole si applicano solo all'utente corrente. Non puoi creare una regola di blocco da messaggi presenti su account condivisi.
:::

Puoi ottenere lo stesso risultato anche con il pulsante **Indesiderato**. WebTop ti chiede se vuoi bloccare il mittente o spostare solo il messaggio.

Scegli **Sì, blocca** per creare la regola e spostare il messaggio. Scegli **No, sposta soltanto** per spostare il messaggio senza creare alcuna regola.

# WebTop-5: consigli di utilizzo del modulo Attività

Usa i filtri rapidi nella barra degli strumenti per passare tra le viste più comuni delle attività.

Il modulo supporta anche attività ricorrenti, sotto-attività collegate e ricerche multiple.

## Filtri rapidi

La barra sopra la griglia include i filtri per le attività di oggi, i prossimi 7 giorni, le attività non iniziate, quelle in ritardo, quelle completate e altre viste rapide.

## Attività ricorrenti

Usa le attività ricorrenti quando devi ripetere la stessa attività con una cadenza definita.

## Sotto-attività collegate

Puoi collegare sotto-attività per tenere insieme lavori correlati.

## Ricerche multiple

Salva o ripeti ricerche diverse quando devi lavorare su più insiemi di attività.

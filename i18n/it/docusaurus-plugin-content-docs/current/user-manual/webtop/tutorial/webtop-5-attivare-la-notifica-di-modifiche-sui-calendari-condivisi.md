# WebTop-5: attivare la notifica di modifiche sui calendari condivisi

Puoi abilitare le notifiche quando un utente modifica eventi su un calendario personale condiviso.

Il proprietario del calendario deve farlo dalle opzioni del calendario. Fai clic destro sul calendario e scegli **Edit calendar**.

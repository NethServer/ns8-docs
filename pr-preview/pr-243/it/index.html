Documentazione di NethServer 8

NethServer 8 è un application server progettato per piccoli uffici e medie imprese

Applicazioni modulari

NethServer è un orchestratore di container che permette di installare e gestire le applicazioni tramite una comoda interfaccia web.

Cluster multi-nodo

Ospita le applicazioni su una singola macchina o distribuisce il carico su piu nodi, pronti per il cloud ibrido.

Open Source

NethServer e Open Source e supportato da una comunità vivace.

# WebTop-5: Tips for using the Activities module

Use the quick filters in the toolbar to switch between the most common activity views.

The module also supports recurring activities, linked subtasks, and multiple searches.

## Quick filters

The toolbar above the grid includes filters for today's activities, the next 7 days, not started items, overdue items, completed items, and more.

## Recurring activities

Use recurring activities when you need to repeat the same task on a schedule.

## Linked subtasks

You can connect subtasks to keep related work grouped together.

## Multiple searches

Save or repeat different searches when you need to work on several activity sets.

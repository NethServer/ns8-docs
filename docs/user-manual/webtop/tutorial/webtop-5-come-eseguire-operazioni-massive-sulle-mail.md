# WebTop-5: Perform bulk mail operations

WebTop uses an infinite grid for mail, so mass actions such as moving a whole folder can be less obvious.

To move the full content of a folder:

1. Select the folder so its messages appear in the grid.
2. Right-click one message and choose **Operations** → **Execute operation**.
3. Select the destination folder.
4. Enable **Execute on entire result** and click **Move**.

If you want to move only some messages, search for them first and then run the operation on the search result.

The same workflow also applies to **Copy**, **Delete**, and **Archive locally**.
